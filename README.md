# 强化学习数学基础 · 交互式课程

一个关于强化学习核心概念与算法的交互式教学网站。

## 课程结构

全书共 10 章，均计划制作成交互式网页：

1. 第 1 章 基本概念
2. 第 2 章 状态值与贝尔曼方程
3. 第 3 章 最优状态值与贝尔曼最优方程
4. 第 4 章 值迭代与策略迭代
5. 第 5 章 蒙特卡洛方法
6. 第 6 章 随机逼近
7. 第 7 章 时序差分方法
8. 第 8 章 值函数方法
9. 第 9 章 策略梯度方法
10. 第 10 章 Actor-Critic 方法

## 技术栈

- React 19 + TypeScript
- Vite
- Tailwind CSS + shadcn/ui
- KaTeX（公式渲染）
- SVG/D3.js（交互式图表）

## 本地开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```
